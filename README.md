  Uma aplicação web para solicitação de reembolso de despesas, desenvolvida com React, TypeScript e Vite.

  ![alt text](src/assets/notes/managerHeader.png)

## Descrição

O Refund App permite que funcionários solicitem reembolso de despesas de forma simples e intuitiva. Os usuários podem preencher um formulário com os dados da despesa, selecionar uma categoria, anexar comprovantes e enviar a solicitação.

## Funcionalidades

- **Formulário de Reembolso**: Campos para nome da solicitação, categoria, valor e upload de arquivo.
- **Autenticação**: Páginas de login e cadastro.
- **Navegação**: Rotas protegidas para funcionários e gestores.
- **Interface Responsiva**: Design moderno com Tailwind CSS.

## Tecnologias Utilizadas

- **React**: Biblioteca para construção de interfaces.
- **TypeScript**: Tipagem estática para JavaScript.
- **Vite**: Ferramenta de build rápida.
- **Tailwind CSS**: Framework CSS utilitário.
- **React Router**: Roteamento para aplicações React.
- **ESLint**: Linting para manter a qualidade do código.
